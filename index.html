🌍 Що відбувається у світі AI

  • Google запустив Gemini 3.1 Pro (лютий 2026) з фокусом на складні задачі, покращене reasoning та агентні сценарії для розробників і бізнесу.
  • OpenAI оновлює API-лінійку: триває перехід на нові моделі та планові deprecations старих snapshot-моделей — командам важливо регулярно оновлювати інтеграції.
  • Ринок зсувається від "демо AI" до операційного AI: ключовий тренд — інтеграція агентів у реальні процеси (продажі, підтримка, документообіг, аналітика).

Джерела: Google Blog, OpenAI API Deprecations

🧱 Що відбувається в Oracle APEX

  • Oracle APEX 24.2 (січень 2025) посилив AI-можливості: RAG-дані для AI-викликів, генерація тексту через Dynamic Action, покращення AI-конфігурації.
  • Додано підтримку векторного пошуку (Oracle Database 23ai), що важливо для enterprise RAG-сценаріїв.
  • APEX рухається в бік AI-native low-code: швидка розробка + надійна Oracle-архітектура для продакшену.

Джерело: Oracle APEX Release Notes

🇺🇦 Що відбувається в Україні щодо заміни 1C

  • В Україні оприлюднено перелік ПЗ, забороненого для визначених категорій організацій, де згадуються 1C та BAS (з 6 січня 2026).
  • Для бізнесу це прискорює попит на безпечні альтернативи та міграційні проєкти з російського ПЗ.
  • Зростає інтерес до нових платформ автоматизації з прозорою архітектурою, аудитом та інтеграцією AI в бізнес-процеси.

Джерела: ProIT, Speka
